    Autenticación

    Autenticación

    /api/authenticate

    POST Accede con identificador de usuario y contraseña

    Se solicita un JSON Web Token de acceso para que un usuario pueda operar con el resto de API. La solicitud incluye el identificador de usuario y la contraseña.

    Cuerpo de la petición

    Parámetro Descripción
    username Nombre del usuario
    password Contraseña del usuario

    Ejemplo:

    {
  "username": "usuario",
  "password": "contraseña"
}

    Respuestas

    Código Descripción
    200 Se ha obtenido un JSON Web Token de acceso
    401 No autorizado
    200 OK

    /api/authenticate devuelve:

    Cabecera Descripción
    Authorization Bearer + JSON Web Token
    Propiedad Descripción
    id_token JSON Web Token

    Respuesta ejemplo:

    {
  "id_token": "eyJhbGciOiJ..."
}
    401 Unauthorized

    /api/authenticate devuelve una respuesta sin cuerpo.
